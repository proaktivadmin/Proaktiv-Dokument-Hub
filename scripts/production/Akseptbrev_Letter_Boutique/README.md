# Akseptbrev Letter — Boutique Style

Thank-you letters for acceptance of purchase offer. **Separate templates for buyer (kjøper) and seller (selger).** All designs are single-column letter layout — no split/two-column layout.

## Files

| File | Recipient | Design |
|------|-----------|--------|
| `kjoper_v01_classic.html` | Kjøper | Classic — fysisk kontraktsmøte |
| `kjoper_v02_minimal.html` | Kjøper | Minimal — same flow as v01 |
| `kjoper_v03_digital.html` | Kjøper | Digital signering — vedlagt kjøpekontrakt, BankID, markedsføringsreservasjon, boligkjøperforsikring |
| `selger_v01_classic.html` | Selger | Classic — fysisk kontraktsmøte, oppgjørsskjema |
| `selger_v02_minimal.html` | Selger | Minimal — same flow as v01 |
| `selger_v03_digital.html` | Selger | Digital signering — vedlagt kjøpekontrakt, BankID, markedsføringsreservasjon |

## Design Notes

- **v01 classic:** Warmer feel with #FDFCFA background, navy (#272630) detail box, bronze (#BCAB8A) left border on property line and detail box.
- **v02 minimal:** Cleaner, more editorial — white background, light beige (#F8F6F2) detail box, increased line-height and margins.

Both inherit Boutique Hotel typography (Open Sans, Playfair Display) and color palette. Use Mottaker and Avsender resources for address blocks in Vitec.
