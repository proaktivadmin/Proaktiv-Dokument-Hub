## vitec-if Betinget Logikk

### Syntaks
```html
<span vitec-if="uttrykk">Vises hvis sant</span>
<span vitec-if="!uttrykk">Vises hvis usant</span>
```

### Operatorer

| Operator | Beskrivelse | Eksempel |
|----------|-------------|----------|
| `==` | Lik | `Model.eiendom.eieform == "Andel"` |
| `!=` | Ulik | `Model.eiendom.eieform != "Andel"` |
| `>`, `<`, `>=`, `<=` | Sammenligning | `Model.pris > 5000000` |
| `.Contains("x")` | Inneholder streng | `Model.oppdrag.type.Contains("tleie")` |
| `.StartsWith("x")` | Starter med | `Model.ref.StartsWith("2024")` |
| `&&` | Og | `Model.a == "x" && Model.b == "y"` |
| `||` | Eller | `Model.a == "x" || Model.b == "y"` |
| `!` | Negasjon | `!Model.oppdrag.type.Contains("tleie")` |

### Eksempler

#### Salg vs Utleie
```html
<span vitec-if="!Model.oppdrag.type.Contains(&quot;tleie&quot;)">Selger</span>
<span vitec-if="Model.oppdrag.type.Contains(&quot;tleie&quot;)">Utleier</span>
```

#### Eieform
```html
<span vitec-if="Model.eiendom.eieform == &quot;Andel&quot;">andel i borettslag</span>
<span vitec-if="Model.eiendom.eieform != &quot;Andel&quot;">fast eiendom</span>
```

### Escape-tegn i HTML
Bruk `&quot;` for anførselstegn inne i HTML-attributter.

### Norske tegn (æ, ø, å)
Norske tegn fungerer direkte i `vitec-if`:
```html
vitec-if="Model.eieform == &quot;Bløtt&quot;"
```

## vitec-foreach Iterasjon

### Syntaks
```html
<element vitec-foreach="item in Model.collection">
```

```html
<div vitec-foreach="s in Model.selgere">
```

```html
<table>
```

## Beste Praksis

1. **Wrap alt i `#vitecTemplate`** for konsistent styling
2. **Inkluder stilark** med `vitec-template="resource:Vitec Stilark"`
3. **Bruk A4-dimensjoner** (21cm bredde, 1cm padding)
4. **Escape anførselstegn** med `&quot;` i HTML-attributter
5. **Test betingelser** grundig med forskjellige oppdragstyper
6. **Hold bunntekst-høyde** på 1.5cm for konsistens
7. **Bruk `::before` pseudo-elements** for flytende labels
