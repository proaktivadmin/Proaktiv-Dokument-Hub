# Token Audit

Generate a lightweight efficiency report to reduce token waste and rework loops.

## Command

```bash
python scripts/tools/token_efficiency_audit.py
```

## Output

Creates:

`scripts/qa_artifacts/TOKEN-AUDIT-{YYYY-MM-DD}.md`

## Use Cases

- End-of-task retrospective
- Before/after major bugfix sessions
- Pre-deploy process quality check with `/deploy-gates`
